About Me

I'm Rabbit Stoddard, and I do all the things, and also none of the things. I like chill, not writing bios, and you. Sometimes, I read tarot cards, run/play LARPS, create conlangs, illuminate manuscripts, and engage in other secondary world fiction.

Currently, I am doing this coding bootcamp thang, so that I can turn some of these ideas into browser games using javascript and Twine, as well as understand more of the development cycle and needs of developers. I am employed as a SAFE4 Agile Certified Scrum Master presently, which means I help make sure that coders can spend their time coding, instead of doing paperwork or answering questions about when they will be done coding. It is a fun thing, and I enjoy it, but there is always more to learn.
